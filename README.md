# Countertop Studio n8n Nodes

Official Countertop Studio integration for n8n.

For more detailed information and full documentation, please visit our documentation page:
http://docs.countertop.app

## Installation

In n8n:

Settings → Community Nodes → Install

Package name:

n8n-nodes-countertop-studio

## Nodes

* Countertop Studio Normalize Deal
* Countertop Studio Send Deal
* Countertop Studio Validate Deal
